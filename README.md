# dotfiles

## For Neovim,

Requirements:
* [Neovim](https://github.com/neovim/neovim) version >= 0.5.0
* [Nerd Fonts](https://www.nerdfonts.com/)
* Python3, pip, pynvim
* Nodejs version >= v14, npm
* [Vim-plug](https://github.com/junegunn/vim-plug) for plugins

Clone this repo and run `nvimScript.sh`.
