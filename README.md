* 512GB NVME

* Windows 11 Pro

* [gnome-terminal](https://gitlab.gnome.org/GNOME/gnome-terminal) + [zsh](https://ohmyz.sh/) + [p10k](https://github.com/romkatv/powerlevel10k)
